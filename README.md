## Fitur
### Login
- Detail: Halaman login sederhana (username/password hardcoded: user/1234). Status login disimpan di SharedPreferences key `isLoggedIn`.

### Light/Dark Mode
- Preferensi tema disimpan di SharedPreferences key `isDarkMode`. Saat aplikasi dibuka, tema diterapkan sesuai preferensi.

### MySQL
- Implementasi My SQL menggunakan PHP My Admin

### SharedPreferences
- Digunakan untuk:
  - menyimpan status login (`isLoggedIn`)
  - menyimpan preferensi tema (`isDarkMode`)
  - menyimpan nama item terakhir (`last_added_item`)
